# NestJS
Nestjs tutorial.. For study 2020.07.23

---
## Download
```
npm i -g @nestjs/cli
```

## Start
```
nest new nest-next-typescript
```

### Directory
```
src
  - app.controller.ts
  - app.service.ts
  - app.module.ts
  - main.ts
 ```
 Controller
 컨트롤러
 
 Service
 간단한 텍스트 요청을 controller에 전달
 
 Module
 application 내 모듈
 
 Main
 application 엔트리 파일
